# Projeto de Gestão de contratos salariais de funcionários

## Descrição
Essse é um sistema onde o usuário coloca seu departamento, e depois coloca os dados do funcionário (nível, nome e salário base).
Após isso o usuário coloca a quantidade de contratos avulsos para o funcionário, e de acordo com os dados de cada contrato (salário por hora, data e horas trabalhadas) o salário total
do funcionário é exibido na data que o usuário rescolher.

## Funcionalidades
- Cadastro de funcionário
- Gestão de salário do funcionário
- Consulta de datas exatas para retorno de dados do funcionário

## Tecnologias
- Java
- Classe enum
- Encapsulamento
- Métodos

## Como executar o projeto

### Pré-requisitos
- Java 17 ou superior instalado
- Execute os comandos dentro da pasta src
- Os nomes dos pacotes devem coincidir com a estrutura de pastas

### Opção 1: Pelo terminal (CMD / Git Bash)

```bash
1. Clone o repositório
git clone <https://github.com/JoaoLucas280/Java-Exercises.git>

2. Acesse a pasta do projeto
cd Java-Exercises/Exercicios Java

3. Entre na pasta src
cd src

4. Compile os arquivos
javac Object_Orientation\Enumeration\EmployeeProject\Application\*.java Object_Orientation\Enumeration\EmployeeProject\Entities\*.java Object_Orientation\Enumeration\EmployeeProject\Entities_enum\*.java

5. Execute o programa
java Object_Orientation.Enumeration.EmployeeProject.Application.Program

```
