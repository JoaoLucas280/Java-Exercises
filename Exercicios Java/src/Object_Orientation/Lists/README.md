# Projeto Gestor de Salário de 1 ou mais funcionários

## Descrição
O projeto consiste em um gestor da quantidade escolhida de funcionários que o usuário coloca. O controle é feito com Id e o nome do funcionário e é guardado em uma lista 
Exemplo: Se o usuário colocar 3 funcionários, o programa pede o id e nome desses 3 funcionários. 
Além disso, o ponto principal do projeto é a fácil atualização salarial desses funcionários, onde é feito uma escolha se vai ser uma redução ou aumento, depois de escolhido,
o programa pergunta o valor para o funcionário e guarda na lista. Após todo esse processo, o programa retorna uma lista atualizada de todos os funcionários.

## Funcionalidades
- Manipulação de vários funcionários
- Manipulação de salário de maneira fácil
- Lista simples atualizada de cada funcionário
- Redução ou aumento salarial pra cada funcionário

- Lists e ArrayLists
- Encapsulamento
- Métodos
